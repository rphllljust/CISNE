Ativacao CISNE

Fluxo guiado para validar e ativar os modulos do sistema sem alterar o que ja esta implantado.

PASSO 1
Validar scripts

Confere se o activator esta carregado e pronto para uso.

PASSO 2
Ativar sistema

Inicializa os modulos e deixa API global disponivel.

PASSO 3
Rodar testes

Executa verificacoes de modulos, SLA e permissoes.

PASSO 4
Concluir

Salva status local de ativacao concluida.

Aguardando execucao.
